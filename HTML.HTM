A Basic Html Code Reference

The Attibutes Reference

The Align Attribute Can Have A Value Of Left or Center or Right
The Size Attribute Can Have A Value Of 1 to 7 or +1 to +7 or -1 to -7
The Color Attribute Can Have A Value Of A Six Digit Hex Number Form #000000 to #ffffff
The Face Attribute Can Have A Value Of Font Names
The Scr Attribute Can Have A Value Of A Path To A Image File
The Width Attribute Can Have A Value Of The Image Width Dimension In Pixels
The Height Attribute Can Have A Value Of The Image Height Dimension In Pixels
The Href Attribute Can Have A Value Of A Address or A Path
The Noshade Attribute Can Have A Value Of Nothing
